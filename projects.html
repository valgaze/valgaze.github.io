Projects

Below are some of the applications on which I've worked

Artwalk

http://www.artwalk.info

Data management tool that helps organizations like the Downtown Los Angeles Artwalk manage their art gallery data on their website and mobile mapping applications

  • Built admin system using AngularJS which allowed non-technical userbase to easily manage gallery data
  • Translated user feedback into new features like mobile map web-view and custom email templates

SheTaxis

http://www.shetaxis.com

New York City’s first female-only ridesharing service. This project got quite a lot of attention

  • Wrote driver management and SMS paging system for the SheTaxis taxi service using Angular and Twilio API
  • Managed remote team and integrated web signup with a legacy “black box” SOAP API

SideChalk

http://www.sidechalkapp.com/about

Location-based mobile application that lets you leave behind memories (drawn, written, or audio) at special places

  • Integrated Firebase’s GeoFire plugin so memories added near a user’s locations are displayed in real time
  • Developed authentication flow using oAuth for users to login with their preferred social networks
  • Implemented emoji-based rating system using React so users could anonymously respond to others’ content

SumoTomato

http://bit.ly/sumotomato

Real-time multiplayer game using mobile phones as input devices for a game displayed on a single shared game screen

  • Built out game controller with Phaser gamer engine for multiple players to interact using their smart phones
  • Designed backend to process user touch inputs and update game state in real time using Socket.IO

VacayPay

http://bit.ly/vacaypayapp

Tool that enables friends to easily track and settle expenses during group trips

  • Created currency conversion feature which let users convert expenses from over 150 world currencies
  • Implemented map using Google Maps API so users could connect their expense history to physical locations
